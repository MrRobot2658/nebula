# 邮件 Email

> **Skill ID** `email-skill` ｜ **分类** 渠道 ｜ **文档版本** v0.2

## 一、模块定位

邮件是历史最悠久、也最成熟的营销触达方式之一。在中国市场，邮件尤其是 B2B 营销的核心渠道——它适合承载较长篇幅的内容、专业的方案介绍与系统化的客户培育（Nurturing），并以稳定的可追踪性著称。邮件渠道 Skill（`email-skill`）提供 HTML 邮件模板编辑、批量发送与定时、打开率与点击率追踪、退订管理及发送域名配置等能力，使邮件成为可设计、可批量、可量化的统一触达通道。

本 Skill 对标 Marketo 等成熟的邮件营销平台，解决企业邮件营销"内容做不精、效果看不清、送达保不住"的问题。Nebula 将邮件的打开、点击、退信、退订等行为实时接入事件总线，经 OneID 与客户主档绑定，使每一封邮件的互动都成为可分析的客户信号并驱动后续培育动作；通过 DKIM/SPF 等发送域名配置保障送达率与品牌信誉，退订管理保障合规。邮件与短信、微信生态等渠道协同，承担 B2B 长周期培育与内容触达的职责，是 Nebula 多渠道矩阵中"深度内容触达"的关键一环。

## 二、目标用户与典型场景

- **目标用户**：以邮件做线索培育的 B2B 市场团队、发送会员通讯与活动邀约的 CRM/会员运营、需要批量且可追踪触达的营销运营。
- **典型场景**：
  - 市场团队用 HTML 模板编辑器制作专业的产品/活动邮件，批量发送或定时下发给目标客户群。
  - 通过打开率与链接点击率追踪邮件互动，识别高意向客户并触发销售跟进或下一步培育。
  - 配置 DKIM/SPF 发送域名并维护退订列表，保障邮件送达率、品牌信誉与合规。

## 三、核心能力

| 能力 | 说明 |
| --- | --- |
| HTML 邮件模板编辑器 | 支持设计与编辑 HTML 邮件模板，制作专业、品牌化的邮件内容。 |
| 批量发送与定时 | 支持向目标人群批量发送并按设定时间下发，匹配营销节奏。 |
| 打开率追踪 | 追踪邮件被打开的情况，量化触达的初步效果。 |
| 链接点击率追踪 | 追踪邮件内链接的点击，识别客户兴趣与高意向行为。 |
| 退订管理 | 管理退订请求与退订列表，保障合规并维护发送健康度。 |
| 发送域名配置（DKIM/SPF） | 配置 DKIM/SPF 等域名认证，提升送达率与防伪信誉。 |

## 四、ChatUI 交互

邮件以批量下发为主，客户对邮件的回复或互动通过追踪与回执进入 ChatUI 中栏，运营可在统一界面查看邮件触达与互动情况；右栏客户卡片展示该客户的邮箱地址、邮件订阅状态与历史打开/点击记录，并经 OneID 与其跨渠道画像融合，帮助运营判断客户的内容偏好与意向阶段。中栏 AI 建议区可基于打开/点击行为提示客户的兴趣信号、推荐合适的后续内容或跟进动作，运营一键采纳。右栏活动区按时间线展示该客户的邮件打开、点击、退信、退订等事件，使邮件培育效果在统一视图中清晰可见。

## 五、触发事件

| 事件 | 触发时机 |
| --- | --- |
| `email_opened` | 客户打开邮件时触发。 |
| `email_clicked` | 客户点击邮件内链接时触发。 |
| `email_bounced` | 邮件投递失败、产生退信时触发。 |
| `email_unsubscribed` | 客户通过邮件退订时触发。 |

## 七、关键指标

- 送达率：衡量邮件成功进入收件箱的比例，受域名认证与发送信誉影响。
- 打开率：衡量邮件标题与触达时机的吸引力。
- 点击率（CTR）：衡量邮件内容引导客户行动的有效性。
- 退信率：衡量邮箱列表质量与发送健康度。
- 退订率：衡量内容相关性与触达频率对客户体验的影响。

## 八、依赖与边界

- **依赖**：统一数据层（CDP / OneID / 事件总线）；邮件发送服务（SMTP/ESP）与 DKIM/SPF 域名认证用于批量投递与追踪。
- **边界**：本 Skill 负责邮件渠道的发送、追踪与互动事件采集及域名/退订管理，不负责邮件创意与文案内容的策划生产（交由内容模块），也不负责跨渠道自动化培育旅程的编排与策略设计（交由活动/自动化模块）。
