# 官网 Website

> **Skill ID** `website-skill` ｜ **分类** 渠道 ｜ **文档版本** v0.2

## 一、模块定位

品牌官网与营销落地页是企业最核心的自有数字资产，不依附于任何第三方平台，是品牌掌握流量入口、沉淀线索与塑造专业形象的主阵地。对 B2B 企业而言，官网更是线索获取（Leads）的关键源头；对 DTC 与零售品牌而言，落地页则是广告投放承接转化的"最后一公里"。官网渠道 Skill（`website-skill`）提供从建站、落地页、表单、SEO 到在线客服、访客追踪与 ChatUI 嵌入的一体化能力，把官网从静态展示页升级为可识别访客、可收集线索、可实时对话的智能营销载体。

本 Skill 对标销售易、纷享销客等的官网营销/获客模块，解决企业官网"流量来了却抓不住、访客行为看不见"的痛点。Nebula 通过访客追踪识别匿名访客并记录行为轨迹，表单提交与线索生成实时写入事件总线，经 OneID 在客户从匿名到实名的旅程中持续归并身份；在线客服与嵌入的 Nebula ChatUI 由原生 AI Agent 驱动，使官网访客的每一次浏览、咨询、留资都成为可运营、可归因的客户事件。

## 二、目标用户与典型场景

- **目标用户**：以官网获客为核心的 B2B 市场团队、需要快速搭建营销落地页的增长/活动运营、依托官网在线客服承接咨询的售前团队。
- **典型场景**：
  - 市场团队用拖拽式建站快速搭建活动落地页并配置 A/B 测试，对比不同版本的留资转化效果。
  - 访客填写官网表单后，线索被智能分发给对应销售，并实时进入客户主档触发跟进流程。
  - 匿名访客在官网浏览多页后通过悬浮客服或嵌入的 ChatUI 发起咨询，AI 自动应答并完成身份识别与线索沉淀。

## 三、核心能力

| 能力 | 说明 |
| --- | --- |
| 建站系统 | 提供拖拽式官网搭建与模板库，让运营无需开发即可快速上线品牌官网。 |
| 落地页 | 支持营销活动独立落地页搭建并内置 A/B 测试，优化投放承接的转化效率。 |
| 表单 | 提供线索收集表单与智能分发，把访客留资自动路由到对应跟进人。 |
| SEO | 支持页面优化、站点地图与结构化数据，提升官网的搜索可见性与自然流量。 |
| 在线客服 | 提供官网悬浮客服与 AI 自动回复，即时承接访客咨询。 |
| 访客追踪 | 识别匿名访客并记录其浏览行为，构建从匿名到实名的访客画像。 |
| 聊天插件 | 将 Nebula ChatUI 嵌入官网，使官网对话与统一会话体系无缝打通。 |

## 四、ChatUI 交互

访客通过官网悬浮客服或嵌入的 Nebula 聊天插件发起的对话，实时进入 ChatUI 中栏会话流，运营在统一界面即可接待官网访客，AI 自动回复兜底常见问题；右栏客户卡片展示该访客的来源页面、浏览轨迹与（在留资后的）线索信息，并经 OneID 把匿名访客与已知客户身份逐步归并，呈现从访问到留资的完整路径。中栏 AI 建议区结合访客浏览行为与意图给出回复话术或留资引导建议，运营一键采纳。右栏活动区展示该访客的页面访问、表单提交、对话发起、线索生成等事件时间线，让官网获客全程可视、可归因。

## 五、触发事件

| 事件 | 触发时机 |
| --- | --- |
| `form_submitted` | 访客提交官网或落地页表单时触发。 |
| `visit_recorded` | 访客访问页面、产生浏览行为被记录时触发。 |
| `chat_started` | 访客通过在线客服或嵌入 ChatUI 发起对话时触发。 |
| `lead_generated` | 访客行为满足线索生成条件、产生新线索时触发。 |

## 七、关键指标

- 官网/落地页访问量与跳出率：衡量流量规模与页面吸引力。
- 表单转化率（留资率）：衡量访客到线索的转化效率。
- 线索数量与线索质量分布：衡量获客的规模与有效性。
- A/B 测试胜出版本转化提升：衡量落地页优化效果。
- 在线客服会话率与响应时长：衡量官网对话承接能力。

## 八、依赖与边界

- **依赖**：统一数据层（CDP / OneID / 事件总线）；官网/落地页托管环境与前端埋点、访客追踪能力用于行为采集与身份识别。
- **边界**：本 Skill 负责官网渠道的搭建、访客行为与线索事件采集及对话承接，不负责营销文案与创意内容生产（交由内容模块），也不负责线索后续的跨渠道自动化培育旅程编排（交由活动/自动化模块）。
