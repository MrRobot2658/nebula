常见问题

Nebula 和微盟/有赞/尘锋有什么不同？

微盟和有赞偏向社交电商（商城+分销），尘锋偏向 SCRM（企微+会话存档）。Nebula 是一个通用的 Marketing Automation Agent，以 ChatUI 统一交互，Skill 插件化架构可以覆盖以上所有场景。核心差异在于：开放插件生态 + 原生 AI 引擎 + 统一 ChatUI。

Skill 需要自己开发吗？

20 个核心 Skill（9 渠道 + 5 内容 + 6 活动）预装开箱即用。只有特殊定制才需要 SDK 开发。

ChatUI 能胜任复杂配置吗？

日常操作在 ChatUI 中完成。复杂配置（自动化画布、评分模型、报表）弹出侧滑面板或独立页面。

同一客户多渠道怎么处理？

OneID 自动匹配。微信、小程序、企微、官网上的同一客户自动识别合并。

小微企业适用吗？

适用。Skill 按需加载，最小可只装 3 个插件（微信+模板库+自动化）。ChatUI 零学习成本。

消息频率限制？

内置频率控制器，遵守各渠道限制。超限自动排队。

数据安全？

全程加密，渠道 Skill 仅持临时凭证。支持私有化部署。

和第三方系统对接？

标准 REST API + Webhook + Skill SDK。
