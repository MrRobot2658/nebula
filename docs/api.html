API 接口文档

RESTful API，Base URL: https://api.nebula.ai/v1

认证

Authorization: Bearer ***

消息 API

POST/v1/messages/send
发送消息到指定渠道
{"channel":"wechat","customer_id":"wx_123","content":"您好，有什么可以帮您？","template_id":"tpl_welcome"}
GET/v1/messages?customer_id={id}&limit=20
查询消息历史

客户 API

GET/v1/customers/{id}
客户详情（含画像/标签/评分/活动）
GET/v1/customers/search?q={keyword}&tags=high-intent
搜索客户
PATCH/v1/customers/{id}/tags
更新标签

活动 / 自动化 / 评分

POST/v1/campaigns
创建活动
POST/v1/automations
创建自动化规则
GET/v1/customers/{id}/score
获取客户评分历史
POST/v1/customers/{id}/score/adjust
手动调整评分
