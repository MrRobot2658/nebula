Nebula 星云

一款面向中国市场的 Marketing Automation Agent。以 ChatUI 为核心交互，9 个渠道 Skill、5 个内容 Skill、6 个活动 Skill 覆盖企业全域营销。

产品定位

Nebula 借鉴 Marketo 的能力模型，但针对中国市场做了三个关键改变：

  • 交互方式：ChatUI 聊天界面替代传统表单后台
  • 渠道覆盖：微信生态（个人/企微/公众号/小程序）+ 官网 + 飞书/钉钉/短信/邮件
  • 架构：渠道/内容/活动全部以 Skill 插件接入，可插拔可组合

对比国内同类产品（微盟、有赞、尘锋、销售易等），Nebula 的核心差异在于 ChatUI 统一交互 + Agent 原生 AI 能力 + 开放 Skill 生态，而非传统 SaaS 的菜单-表单-表格范式。
