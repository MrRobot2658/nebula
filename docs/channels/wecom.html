企业微信 WeCom wecom-skill

企业微信渠道，SCRM 核心场景。

企业微信是中国 B2C 客户运营的核心载体。Nebula 的 WeCom Skill 对标尘锋、探马等 SCRM 产品能力，支持客户联系、群管理、标签体系、会话存档和群发助手，是企业精细化运营客户关系的必备工具。

能力列表

  • 客户联系：一对一聊天、群聊
  • 客户群管理：创建/管理/裂变
  • 标签体系：自动化打标签、标签分组
  • 会话存档：合规保存聊天记录
  • 群发助手：批量发送消息给客户/群
  • 渠道活码：不同渠道引流的客户自动打标签
  • 离职继承：客户关系自动转移
  • 侧边栏：客服快捷回复、客户信息卡片

触发事件

  • message_received
  • customer_added
  • group_joined
  • tag_changed

配置参数

{ "api_key": "your_wecom_api_key", "webhook_url": "https://your-domain.com/webhook", "enabled": true }
