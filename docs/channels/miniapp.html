小程序 Mini Program miniapp-skill

微信/支付宝小程序渠道。

小程序是中国移动互联网的核心商业载体。Nebula 的 Mini Program Skill 对标有赞微商城、微盟智慧零售的能力，支持小程序商城搭建、模板消息订阅、支付闭环和数据分析，是 DTC 品牌转化的关键渠道。

能力列表

  • 小程序商城：商品展示/购物车/下单支付
  • 模板消息订阅：下单提醒、物流更新、活动通知
  • 客服消息：小程序内客服会话
  • 用户授权：获取手机号、头像昵称
  • 支付：微信支付/支付宝集成
  • 分享裂变：拼团、砍价、分销
  • 数据分析：访问/转化/复购数据

触发事件

  • order_placed
  • payment_success
  • customer_service_msg
  • share_clicked

配置参数

{ "api_key": "your_miniapp_api_key", "webhook_url": "https://your-domain.com/webhook", "enabled": true }
