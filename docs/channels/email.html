邮件 Email email-skill

邮件渠道，传统营销触达方式。

邮件是 B2B 营销的核心渠道。Nebula 的 Email Skill 对标 Marketo 的邮件营销能力，支持 HTML 模板设计、批量发送、打开/点击追踪。

能力列表

  • HTML 邮件模板编辑器
  • 批量发送与定时
  • 打开率追踪
  • 链接点击率追踪
  • 退订管理
  • 发送域名配置（DKIM/SPF）

触发事件

  • email_opened
  • email_clicked
  • email_bounced
  • email_unsubscribed

配置参数

{ "api_key": "your_email_api_key", "webhook_url": "https://your-domain.com/webhook", "enabled": true }
