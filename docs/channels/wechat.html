微信 WeChat wechat-skill

个人微信渠道，通过微信协议客户端对接。

微信是中国最大的社交平台（月活 13 亿），个人微信渠道是品牌与消费者建立直接联系的第一触点。Nebula 的微信 Skill 支持消息收发、好友管理、朋友圈互动和支付能力，覆盖品牌与客户之间的日常沟通场景。

能力列表

  • 消息收发：文本、图片、语音、视频、链接
  • 好友管理：添加/删除好友、备注标签
  • 朋友圈：发布品牌内容、查看客户互动
  • 支付：收款码、红包、转账
  • 群聊管理：品牌群运营
  • 朋友圈广告：定向投放客户朋友圈

触发事件

  • message_received
  • friend_added
  • moments_interacted
  • payment_received

配置参数

{ "api_key": "your_wechat_api_key", "webhook_url": "https://your-domain.com/webhook", "enabled": true }
