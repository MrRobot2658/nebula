Nebula / 快速入门

快速入门

5 分钟跑通第一个自动化营销流程。

1

安装渠道 Skill

设置 → Skill 管理 → 添加 Skill → 选择 wechat-skill → 填入 API Key 和 Webhook URL → 启用

2

创建内容模板

在模板库中创建消息模板，支持变量注入：

「欢迎光临 {{customer.name}}！我们为您准备了 {{campaign.discount}} 优惠。」
3

配置自动化规则

在 ChatUI 右侧面板配置：当新客户发来第一条消息时，自动发送欢迎模板并打上「已互动」标签。

4

完成

下次客户通过微信联系品牌时，Nebula 自动把对话路由到 ChatUI，AI 分析客户意图，自动化规则自动触发。
