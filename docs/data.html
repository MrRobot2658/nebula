Nebula / 统一数据层

统一数据层

CDP + OneID + 事件总线，所有 Skill 共享的数据基础设施。

CDP（客户数据平台）

集成所有渠道的客户数据，对标创略科技 CDP 和销售易 CDP 的能力：

  • 行为数据：浏览、点击、购买、退款、客服咨询
  • 属性数据：年龄、地域、行业、职级、企业信息
  • 互动数据：咨询、投诉、好评、会议参与
  • 交易数据：订单金额、频次、品类偏好、客单价

OneID（身份统一）

  • ID 映射：微信 OpenID / 企微 ExternalUserID / 小程序 UnionID / 手机号 / 邮箱 / 设备 ID
  • 合并策略：确定性匹配（手机号/UnionID）+ 概率匹配（行为相似度）
  • 冲突解决：优先级规则 + 人工合并

事件总线

  • 发布/订阅模式，至少一次投递
  • 事件持久化至数据库，支持回放
  • 跨 Skill 消息延迟 < 100ms
