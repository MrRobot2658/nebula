Nebula / 核心概念

核心概念

Skill（插件）

Nebula 的基本功能单元。每个 Skill 包含四部分：

  • Handler — 业务逻辑
  • UI Component — ChatUI 中的展示组件
  • Config Schema — JSON Schema 配置
  • Events — 发出和监听的事件

ChatUI 三栏布局

  • 左栏（260px）：导航面板，渠道/内容/活动分区
  • 中栏（700px）：对话主界面 + AI 建议 + 快捷回复
  • 右栏（540px）：客户卡片 + 活动状态 + 自动化规则

事件总线

  • message_received / message_sent
  • customer_updated / score_changed
  • action_triggered / schedule_fired

OneID（身份统一）

跨渠道身份自动关联：微信 OpenID → 企微 ExternalUserID → 小程序 UnionID → 手机号 → 邮箱。确定性匹配 + 概率匹配。
