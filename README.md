# Nebula 星云

Marketing Automation Agent — 以 ChatUI 为核心交互，9 渠道/5 内容/6 活动 Skill 插件化架构。

🚀 **在线文档**: [https://nebula-mauve.vercel.app](https://nebula-mauve.vercel.app) (deploying...)

## 文档结构

```
docs/
├── index.html               ← 产品概述
├── quickstart.html          ← 快速入门
├── concepts.html            ← 核心概念
├── channels/                ← 9 个渠道 Skill
├── content/                 ← 5 个内容 Skill
├── activity/                ← 6 个活动 Skill
├── data.html                ← 统一数据层
├── case.html                ← 完整案例
├── api.html                 ← API 接口文档
├── sdk.html                 ← SDK 开发规范
├── faq.html                 ← FAQ + 竞品对比
└── css/style.css            ← 共享样式
```

## 技术栈

- Tailwind CSS (CDN)
- Vanilla HTML
- Vercel (部署)
